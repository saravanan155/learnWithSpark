# Spark mascot assets

Drop the 6 pre-made transparent-background PNGs here (created offline from the locked Spark
reference — see PLAN.md → "Spark mascot"). Until they exist, `<Spark>` shows an emoji placeholder,
so the game still runs.

Expected files (exact names — `Spark.tsx` maps moods to these):

- `spark_curious.png`
- `spark_proud.png`
- `spark_confused.png`
- `spark_excited.png`
- `spark_unsure.png`
- `spark_confident.png`

Rules: same Spark every time, transparent PNG (no scene/props/text), identical framing & size,
centered — so swapping moods never makes Spark jump or resize.
